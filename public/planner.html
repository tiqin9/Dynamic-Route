Order Types

HV (High Value): Standard high-value deliveries. Max 3 packs per flatbed truck.

HVB (High Value Battery): Battery deliveries with return pickups included.

VOR (Vehicle Off Road): Vehicle supplies. Must be delivered to far stops (>100 miles) only after 5 PM.

MRB (Material Return Battery): Battery returns from service centers.

Constraints

  • Max 3 stops per route (layover next day if more needed)
  • Trailer capacity: 14 packs max
  • Flatbed capacity: 8 packs max (HV packs max 3)
  • Route duration: < 24 hours
  • VOR deliveries to far locations (>100 miles): after 5 PM only
  • Departure: 7 AM | Planning deadline: 5 PM

Algorithm

The planner uses:

  • Greedy Assignment: Orders sorted by priority and type, assigned to routes respecting constraints
  • Nearest Neighbor Optimization: Minimizes distance between consecutive stops
  • Constraint Validation: Every route checked against capacity, time, and geographic constraints

Usage Flow

  1. Load Orders: Click "Load Sample Orders" or create custom orders
  2. Set Constraints: Adjust planning parameters as needed
  3. Generate Plan: Click "Generate Optimized Plan" to see route assignments
  4. View Results: See optimized routes with distance/time metrics and map visualization
  5. Download Manifest: Export route details for drivers
